# DTES

DTES in this repository means Deformable Tropical Energy Space.

- It is used as a representation layer for symbolic, geometric, or operator-related state.
- In code, DTES appears mostly in `core/dtes_*` modules and the DTES-braid serialization path.
- It is not treated as a proof object.
- It is not dynamic time warping.
- The TopologicalLM branch uses DTES as part of the episode DSL and token model.
- The operator side also uses DTES-related modules for geometry, graph operators, and spectral-learning experiments.
- In practice, DTES is a modeling convention that links data structures, not a validated theorem.
