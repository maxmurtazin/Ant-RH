# Quickstart

## Main run sequence

```bash
make run-v12
make analyze-gemma
make lab-journal
make study
```

## TopologicalLM branch

```bash
make topo-all
```

This runs:

1. `make topo-train`
2. `make topo-eval`
3. `make topo-report`

## Interactive help

```bash
make help-chat
```

## Literature and docs refresh

```bash
make literature
make docs
```
